# Features - Joypurhat Ramadan Calendar 2026

## 1. Enterprise Mobile UI
- **Mobile-First Design**: Optimized for small screens with an app-shell feel.
- **Glassmorphism**: Modern UI cards with backdrop-blur effects.
- **Color Coding**: Distinct themes for Mercy (Green), Forgiveness (Red), and Salvation (Dark Green).

## 2. Dynamic Countdown
- Real-time countdown to the next event (Sehri or Iftar).
- Automatically detects the current state based on BDT timezone.
- Visual alerts (pulse animation) for upcoming events.

## 3. "Add to Calendar" (Alarm System)
- Users can download ICS files for any specific Ramadan day.
- Events include both Sehri end time and Iftar time.
- Integrated with mobile native calendars (Google Calendar, iCal) to trigger system alarms.

## 4. Accurate Local Timings
- Hardcoded and verified timings specifically for Joypurhat district.
- Supports the full 30-day Ramadan 2026 schedule.

## 5. Performance & SEO
- Server-side rendering for instant load times.
- SEO optimized meta tags for social sharing.
- Secure HTTPS encryption.
